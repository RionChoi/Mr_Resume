## English Version

### 📌 Project Overview

**Mr_Resume** is a premium developer portfolio platform designed with the **Swiss International Style** (Minimalism & Functionalism). It is specifically engineered to meet the high standards of European tech recruitment (Germany, Netherlands, Sweden, UK), focusing on clear communication of technical depth through a "Case Study" approach.

### ✨ Key Features

- **Case Study Layout**: Projects are presented using the _Problem → Solution → Impact_ framework.
- **Data Visualization**: Skill proficiency and technical growth visualized via Recharts.
- **Interactive UI**: Fluid animations powered by Framer Motion and accessible components from Radix UI.
- **Responsive Design**: Seamless experience across mobile, tablet, and desktop.
- **SEO Optimized**: Built with Next.js App Router for best-in-class performance and discoverability.

### 🛠 Tech Stack

- **Framework**: Next.js 16 (App Router)
- **UI Library**: React 19, Radix UI, Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Data Visualization**: Recharts
- **Forms & Validation**: React Hook Form, Zod
- **Typography**: Playfair Display (Headings), DM Sans (Body), JetBrains Mono (Code)

### 🎨 Design Philosophy

Inspired by the **Swiss International Typography Style (Bauhaus-derived)**:

1. **Grid-Based Rigor**: Everything aligns to a vertical/horizontal axis.
2. **Typography First**: Hierarchy is established through font weight and size, not excessive decoration.
3. **Color Palette**: Minimalist White (#FFFFFF) & Noir (#111111) with a signature Swiss Red (#E63946) accent.
4. **Information Density**: Clean spacing with "Sticky" labels for seamless navigation.
